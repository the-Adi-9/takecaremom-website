# Image Assets for Take Care Mom Website

## Required Images

Place the following images in this directory:

### Logo Images
- `logo.png` - Logo for header navigation (square, ~100x100px)
- `logo-hero.png` - Large logo with hands for hero section (from PDF page 1/2)

### App Screenshots
- `app-mockup.png` - App screenshot showing the SuperApp (from PDF page 7)

### Community Screenshots
- `community-mockup.png` - Telegram community screenshot (from PDF page 6)

### Team Photos (in /team folder)
- `team/zulfiya.jpg` - Zulfiya Urunova, CEO
- `team/zhaniya.jpg` - Zhaniya Aitbayeva, CFO

### Optional
- `og-default.jpg` - Open Graph image for social sharing (1200x630px)
- `podcast-cover.png` - Podcast cover image (from PDF page 10)

## Image Guidelines
- Use PNG for logos with transparency
- Use JPG for photos
- Optimize images for web (compress without losing quality)
- Recommended sizes:
  - Logo: 200x200px (will be displayed smaller)
  - Hero logo: 800x800px
  - Mockups: 800px width max
  - Team photos: 400x400px minimum
